📚 小说导出指南 ​

本指南介绍如何为《安乐回响》提供多种格式的导出功能。

🎯 实现的功能 ​

1. 在线打印/导出（已实现）✅ ​

用户体验：

  • 访问任意章节页面
  • 点击"打印/导出 PDF"按钮
  • 使用浏览器打印功能保存为 PDF

特点：

  • ✅ 无需额外依赖
  • ✅ 即时导出
  • ✅ 自动优化打印样式
  • ✅ 支持所有现代浏览器

已添加的文件：

  • .vitepress/theme/index.ts - 主题配置
  • .vitepress/theme/components/PrintButton.vue - 导出按钮组件
  • .vitepress/theme/custom.css - 打印样式优化

2. 完整版 PDF 下载（需要生成） ​

步骤：

步骤 1：生成合并的 Markdown 文件 ​

bash
npm run generate:book

这会生成 安乐回响-完整版.md，包含所有章节。

步骤 2：转换为 PDF ​

有多种方式可以将 Markdown 转换为 PDF：

方式 A：使用 Typora（推荐，最简单） ​
  1. 下载安装 Typora
  2. 打开 安乐回响-完整版.md
  3. 点击 文件 > 导出 > PDF
  4. 保存为 安乐回响-完整版.pdf

优点：

  • 界面友好
  • 样式美观
  • 支持中文
  • 可以自定义主题
方式 B：使用 Pandoc（命令行） ​
bash
# 安装 Pandoc
# macOS
brew install pandoc

# 安装 LaTeX（用于生成 PDF）
brew install --cask mactex-no-gui

# 转换为 PDF
pandoc 安乐回响-完整版.md -o 安乐回响-完整版.pdf \
  --pdf-engine=xelatex \
  -V CJKmainfont="PingFang SC" \
  -V geometry:margin=2cm \
  --toc \
  --toc-depth=2

优点：

  • 自动化
  • 可定制性强
  • 支持批量处理
方式 C：使用在线工具 ​
  1. 访问 Markdown to PDF
  2. 上传 安乐回响-完整版.md
  3. 下载生成的 PDF

优点：

  • 无需安装软件
  • 快速简单
方式 D：使用 VS Code + Markdown PDF 插件 ​
  1. 安装 VS Code 插件：Markdown PDF
  2. 打开 安乐回响-完整版.md
  3. 右键 > Markdown PDF: Export (pdf)

步骤 3：放置 PDF 文件 ​

bash
# 创建下载目录
mkdir -p public/downloads

# 移动 PDF 文件
mv 安乐回响-完整版.pdf public/downloads/

重要： public 目录中的文件会被复制到构建输出的根目录。

📖 其他格式支持 ​

EPUB 格式（电子书） ​

使用 Pandoc 转换：

bash
pandoc 安乐回响-完整版.md -o 安乐回响-完整版.epub \
  --toc \
  --epub-cover-image=cover.jpg \
  --metadata title="安乐回响" \
  --metadata author="zer0fire"

然后放到 public/downloads/ 目录。

MOBI 格式（Kindle） ​

  1. 先生成 EPUB
  2. 使用 Calibre 转换：
bash
# 安装 Calibre
brew install --cask calibre

# 转换
ebook-convert 安乐回响-完整版.epub 安乐回响-完整版.mobi

DOCX 格式（Word） ​

bash
pandoc 安乐回响-完整版.md -o 安乐回响-完整版.docx

🎨 自定义 PDF 样式 ​

添加封面 ​

编辑 scripts/generate-full-book.js，在开头添加封面内容：

javascript
let fullContent = `---
title: 安乐回响
author: zer0fire
---

<div style="text-align: center; padding: 100px 0;">
  <h1 style="font-size: 48px; margin-bottom: 20px;">安乐回响</h1>
  <h2 style="font-size: 24px; color: #666;">数字永生与技术伦理</h2>
  <p style="margin-top: 50px; font-size: 18px;">作者：zer0fire</p>
</div>

<div style="page-break-after: always;"></div>
`;

自定义打印样式 ​

编辑 .vitepress/theme/custom.css 中的 @media print 部分：

css
@media print {
  /* 修改字体大小 */
  body {
    font-size: 14pt;  /* 调整这里 */
  }
  
  /* 修改页边距 */
  @page {
    margin: 3cm;  /* 调整这里 */
  }
  
  /* 添加页眉 */
  @page {
    @top-center {
      content: "安乐回响 - zer0fire";
    }
  }
}

🚀 自动化工作流 ​

GitHub Actions 自动生成 PDF ​

创建 .github/workflows/generate-pdf.yml：

yaml
name: Generate PDF

on:
  push:
    branches: [main]
    paths:
      - '安乐回响-AI/**'

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      
      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: 22
      
      - name: Install Pandoc
        run: |
          sudo apt-get update
          sudo apt-get install -y pandoc texlive-xetex texlive-lang-chinese
      
      - name: Generate full book
        run: npm run generate:book
      
      - name: Convert to PDF
        run: |
          pandoc 安乐回响-完整版.md -o public/downloads/安乐回响-完整版.pdf \
            --pdf-engine=xelatex \
            -V CJKmainfont="Noto Sans CJK SC" \
            --toc
      
      - name: Commit PDF
        run: |
          git config --local user.email "action@github.com"
          git config --local user.name "GitHub Action"
          git add public/downloads/
          git commit -m "Auto-generate PDF" || exit 0
          git push

📱 移动端优化 ​

打印按钮在移动端会自动调整为全宽按钮，提供更好的触摸体验。

🔧 故障排查 ​

问题 1：打印时样式混乱 ​

解决： 检查 .vitepress/theme/custom.css 中的 @media print 样式是否正确加载。

问题 2：中文 PDF 显示乱码 ​

解决： 使用 Pandoc 时指定中文字体：

bash
-V CJKmainfont="PingFang SC"  # macOS
-V CJKmainfont="Microsoft YaHei"  # Windows
-V CJKmainfont="Noto Sans CJK SC"  # Linux

问题 3：PDF 文件太大 ​

解决：

  • 压缩图片
  • 使用 PDF 压缩工具
  • 调整 Pandoc 输出质量
