🚀 快速开始 - 生成 PDF ​

三步完成 PDF 导出功能 ​

第 1 步：生成合并的 Markdown ​

bash
npm run generate:book

✅ 生成文件：安乐回响-完整版.md

第 2 步：转换为 PDF ​

选项 A：Typora（推荐，最简单）⭐ ​

  1. 下载 Typora：https://typora.io/
  2. 打开 安乐回响-完整版.md
  3. 点击：文件 → 导出 → PDF
  4. 保存为：安乐回响-完整版.pdf

选项 B：在线工具（无需安装） ​

  1. 访问：https://www.markdowntopdf.com/
  2. 上传：安乐回响-完整版.md
  3. 下载生成的 PDF
  4. 重命名为：安乐回响-完整版.pdf

选项 C：Pandoc（命令行，适合自动化） ​

bash
# macOS 安装
brew install pandoc
brew install --cask mactex-no-gui

# 转换
pandoc 安乐回响-完整版.md -o 安乐回响-完整版.pdf \
  --pdf-engine=xelatex \
  -V CJKmainfont="PingFang SC" \
  -V geometry:margin=2cm \
  --toc \
  --toc-depth=2

第 3 步：放置 PDF 文件 ​

bash
# 移动 PDF 到下载目录
mv 安乐回响-完整版.pdf public/downloads/

# 验证文件
ls -lh public/downloads/

测试和部署 ​

本地测试 ​

bash
npm run docs:build
npm run docs:preview

访问：http://localhost:4173/demo/ai-novel/

测试：

  • ✅ 首页"下载 PDF"按钮
  • ✅ "导出选项"区域的"下载完整版"按钮
  • ✅ 任意章节的"打印/导出 PDF"按钮

部署到线上 ​

bash
git add .
git commit -m "Add complete PDF version"
git push origin main

等待 GitHub Actions 自动部署完成。

🎉 完成！ ​

现在访问 http://blog.zer0fire.me/ai-novel/

用户可以：

  1. 在线阅读 - 完整的 VitePress 体验
  2. 单章导出 - 点击"打印/导出 PDF"按钮
  3. 下载完整版 - 点击"下载 PDF"按钮

💡 提示 ​

如果 PDF 文件还没生成 ​

下载按钮会显示 404 错误。这是正常的，生成 PDF 后就可以了。

临时方案 ​

如果暂时没有 PDF，可以先注释掉下载按钮：

编辑 index.md，注释掉这一行：

yaml
    # - theme: alt
    #   text: 下载 PDF
    #   link: /downloads/安乐回响-完整版.pdf

编辑 .vitepress/theme/components/PrintButton.vue，注释掉下载按钮部分。

更新 PDF ​

每次更新章节后，重新运行：

bash
npm run generate:book
# 然后重新转换为 PDF
mv 安乐回响-完整版.pdf public/downloads/
git add public/downloads/
git commit -m "Update PDF"
git push
