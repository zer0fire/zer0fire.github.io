📚 导出功能说明 ​

✅ 已实现的功能 ​

你的 VitePress 网站现在支持两种导出方式：

1. 🖨️ 浏览器打印/导出（在线即时） ​

用户操作：

  1. 访问任意章节页面
  2. 点击页面上的"打印/导出 PDF"按钮
  3. 在浏览器打印对话框中选择"另存为 PDF"
  4. 保存到本地

特点：

  • ✅ 无需下载，即时导出
  • ✅ 可以导出单个章节
  • ✅ 自动优化打印样式（隐藏导航、侧边栏等）
  • ✅ 支持自定义页边距、字体大小
  • ✅ 所有现代浏览器都支持

2. 📥 完整版 PDF 下载 ​

用户操作：

  1. 在首页点击"下载 PDF"按钮
  2. 或在首页"导出选项"区域点击"下载完整版"
  3. 直接下载包含所有章节的 PDF 文件

特点：

  • ✅ 一次下载，包含所有章节
  • ✅ 专业排版，带封面和目录
  • ✅ 适合离线阅读
  • ✅ 可以分享给他人

🎨 界面展示 ​

首页 ​

  • Hero 区域有"下载 PDF"按钮
  • "导出选项"区域有详细说明和两个按钮

章节页面 ​

  • 每个章节都可以单独打印导出

🛠️ 你需要做的（生成完整版 PDF） ​

快速开始 ​

  1. 生成合并的 Markdown 文件：

    bash
    npm run generate:book

    这会生成 安乐回响-完整版.md

  2. 转换为 PDF（选择一种方式）：

    方式 A：使用 Typora（最简单，推荐）

    • 下载 Typora
    • 打开 安乐回响-完整版.md
    • 文件 → 导出 → PDF
    • 保存为 安乐回响-完整版.pdf

    方式 B：使用在线工具

    • 访问 https://www.markdowntopdf.com/
    • 上传 安乐回响-完整版.md
    • 下载生成的 PDF

    方式 C：使用 Pandoc（命令行）

    bash
    # macOS 安装
brew install pandoc
brew install --cask mactex-no-gui

# 转换
pandoc 安乐回响-完整版.md -o 安乐回响-完整版.pdf \
  --pdf-engine=xelatex \
  -V CJKmainfont="PingFang SC" \
  -V geometry:margin=2cm \
  --toc

  3. 放置 PDF 文件：

    bash
    mv 安乐回响-完整版.pdf public/downloads/

  4. 测试和部署：

    bash
    # 本地测试
npm run docs:build
npm run docs:preview
# 访问 `http://localhost:4173/demo/ai-novel/` 测试下载链接

# 部署
git add .
git commit -m "Add complete PDF version"
git push origin main

🎯 用户体验流程 ​

场景 1：想快速导出当前章节 ​

  1. 正在阅读第三章
  2. 点击"打印/导出 PDF"按钮
  3. 浏览器打印 → 另存为 PDF
  4. 得到"第三章.pdf"

场景 2：想下载完整版离线阅读 ​

  1. 访问首页
  2. 点击"下载 PDF"或"下载完整版"按钮
  3. 得到"安乐回响-完整版.pdf"（包含所有章节）

场景 3：想在手机上阅读 ​

  1. 下载完整版 PDF
  2. 在手机 PDF 阅读器中打开
  3. 离线阅读

🔧 自定义选项 ​

修改打印样式 ​

编辑 .vitepress/theme/custom.css：

css
@media print {
  /* 修改字体大小 */
  body {
    font-size: 14pt;  /* 默认 12pt */
  }
  
  /* 修改页边距 */
  @page {
    margin: 3cm;  /* 默认 2cm */
  }
  
  /* 添加页眉 */
  @page {
    @top-center {
      content: "安乐回响 - 第" counter(page) "页";
    }
  }
}

修改按钮样式 ​

编辑 .vitepress/theme/components/PrintButton.vue 的 <style> 部分。

添加更多导出格式 ​

编辑 scripts/generate-full-book.js，然后使用 Pandoc 转换：

bash
# EPUB（电子书）
pandoc 安乐回响-完整版.md -o 安乐回响-完整版.epub --toc

# DOCX（Word）
pandoc 安乐回响-完整版.md -o 安乐回响-完整版.docx

# MOBI（Kindle）
ebook-convert 安乐回响-完整版.epub 安乐回响-完整版.mobi

然后都放到 public/downloads/ 目录，并在 PrintButton.vue 中添加下载链接。
